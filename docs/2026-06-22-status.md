## Implemented

* `bin/classroom-archiver.py` exists and is executable.
* CLI flags available:
  * `--creds` required
  * `--output`
  * `--course`
  * `--teacher-email`
  * `--list-classrooms`
  * `--help`
* OAuth Desktop flow implemented with:
  * token cache beside credentials: `.secrets/token.json`
  * fixed callback binding: `localhost:8080`
  * relaxed OAuth scope validation
* Google API scopes:
  * `classroom.courses.readonly`
  * `classroom.courseworkmaterials.readonly`
  * `classroom.student-submissions.me.readonly`
  * `drive.readonly`
* Live scraper supports:
  * `ACTIVE` and `ARCHIVED` course fetching
  * pagination
  * optional `teacherId` filtering via `--teacher-email`
  * courseWork and courseWorkMaterials fetching
  * Drive/link/YouTube/form material mapping into dataclasses
